API Access

Get separate developer tokens for each service. Tokens are valid for 24 hours.

Please sign in or register to access developer APIs.

Identity verification via Firebase is required to obtain secure developer API tokens.

Integrated API Management: Developer tokens are requested seamlessly using your logged-in session. No manual email or name input is required.
Generate Tokens: Navigate to any service's API Access sub-tab to generate tokens for that service.

Clinical Document API - /pdf2abdm

Method Path Auth Description
POST/pdf2abdm🔒 BearerSync: PDF → clinical resources and bundle
POST/pdf2abdm/submit🔒 BearerAsync: upload PDF, returns task_id (202 Accepted)
GET/pdf2abdm/task-status/{id}🔓 OpenPoll async task status (PENDING / SUCCESS / FAILURE)
GET/pdf2abdm/health🔓 OpenService liveness + model status
GET/pdf2abdm/model-health🔓 OpenAI model readiness check
