Run Locally (Offline Mode)

Download a single executable for your OS. No Python, no pip, no cloud APIs. All data stays on your machine.

Requires: NVIDIA GPU with 16 GB+ VRAM (or Apple Silicon Mac with 16 GB+ memory) and ~20 GB free disk space.

Linux x86_64

AppImage (~400 MB)

Ubuntu 20.04+, Fedora 33+

Linux ARM64

Binary (~400 MB)

aarch64 / AWS Graviton

Windows

Portable EXE (~400 MB)

Windows 10 / 11

macOS

Binary (~400 MB)

macOS 13+ (Apple Silicon / Intel)

Setup Steps (All Platforms)

1. Install Ollama

Download from ollama.com/download
Linux: curl -fsSL https://ollama.com/install.sh | sh

2. Pull the LLM Model (~16 GB download)

ollama pull gemma4:26b
Requires NVIDIA GPU with 16 GB+ VRAM, or Apple Silicon with 16 GB+ unified memory.

3. Unzip & Make Executable

Unzip the downloaded file. On Linux/Mac run: chmod +x nhcx-extract*
Linux requires sudo apt install libfuse2 if not already installed.

4. Verify & Run

Linux (x86_64):

./nhcx-extract-1.0.0-x86_64.AppImage check -m gemma4:26b
./nhcx-extract-1.0.0-x86_64.AppImage abdm report.pdf -o result.json -m gemma4:26b
./nhcx-extract-1.0.0-x86_64.AppImage nhcx policy.pdf -o result.json -m gemma4:26b

Linux (ARM64):

./nhcx-extract-linux-aarch64 check -m gemma4:26b
./nhcx-extract-linux-aarch64 abdm report.pdf -o result.json -m gemma4:26b
./nhcx-extract-linux-aarch64 nhcx policy.pdf -o result.json -m gemma4:26b

Windows:

nhcx-extract.exe check -m gemma4:26b
nhcx-extract.exe abdm report.pdf -o result.json -m gemma4:26b
nhcx-extract.exe nhcx policy.pdf -o result.json -m gemma4:26b

macOS:

./nhcx-extract check -m gemma4:26b
./nhcx-extract abdm report.pdf -o result.json -m gemma4:26b
./nhcx-extract nhcx policy.pdf -o result.json -m gemma4:26b

Privacy: All data stays on your machine. Zero cloud API calls. The output JSON contains a complete FHIR R4 Bundle.
